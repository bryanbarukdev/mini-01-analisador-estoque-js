# Analisador de Estoque

![Status](https://img.shields.io/badge/status-em%20desenvolvido-green)
![Licença](https://img.shields.io/badge/license-MIT-green)

## Descrição
Aplicação em JavaScript para processar uma lista de produtos e gerar relatórios de apoio à decisão, como estoque baixo, valor total por categoria e produtos em promoção.

## Tecnologias
- JavaScript
- HTML/CSS (opcional para exibição em tela)

## Funcionalidades
- Filtrar produtos com quantidade abaixo de um limite.
- Calcular valor total de estoque por categoria.
- Listar produtos com preço abaixo do valor definido.

## Como executar
1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador (ou execute com Node.js, se aplicável).
3. Ajuste o array `produtos` e execute as funções.

## Evidências
- Print do `console.table` com resultados.

## Aprendizados
- Uso de `map`, `filter` e `reduce` em cenário real.
- Organização de funções puras para análise de dados.
